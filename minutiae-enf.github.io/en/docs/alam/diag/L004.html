diagnostic Lexing

Malformed duration literal

SeverityError
PhaseLexing
StatusEmitted
Unit outcomeError: unit rejected. Warning: unit may still be published.
Canonical messagemalformed duration literal '<literal>'

Cause

A decimal digit run followed by a duration unit begins one duration token, but the complete token does not preserve strict digit-unit alternation.

A malformed suffix remains part of the same erroneous duration token. It is not split into a shorter valid duration followed by another token.

Example that triggers it

const Duration span = 3hx;

Remedy

Write the duration as one or more complete integer-unit segments.

Supported units are s, m, h, d, and w. For example, use 3h, 1h30m, or 2w3d.
