L002

diagnostic Lexing

Unterminated text literal

SeverityError
PhaseLexing
StatusEmitted
Unit outcomeError: unit rejected. Warning: unit may still be published.
Canonical messageunterminated text literal

Cause

A double-quoted text literal reached the end of the source before a closing double quote was found.

A newline does not terminate a text literal. Scanning continues until a closing quote or the end of the complete source.

Example that triggers it

const Text reason = "missing closing quote;

Remedy

Add the closing double quote.

To include a literal double quote inside the value, escape it as \".
