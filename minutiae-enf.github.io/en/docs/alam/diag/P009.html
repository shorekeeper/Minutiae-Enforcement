diagnostic Parsing

Empty template constraint block

SeverityError
PhaseParsing
StatusEmitted
Unit outcomeError: unit rejected. Warning: unit may still be published.
Canonical messageconstraint block must contain at least one constraint

Cause

A where block closes without containing any constraint expression.

The where clause exists only to impose one or more pure Bool constraints. An empty block has no distinct semantic meaning.

Example that triggers it

template<Measure M>
where {
}
verdict policy = (event) -> Sanction {
    .target = event.subject,
    .cite = { P.1.1 },
    .measure = M,
    .duration = Duration::Permanent,
    .escalation = Escalation::None,
    .annotations = {},
    .attribution = Attribution::System,
    .dry_run = DryRun::InheritSafety,
};

Remedy

Add at least one pure Bool constraint to the where block.

Remove the complete where block when the template has no constraints.
