L008

diagnostic Lexing

Stray vertical bar

SeverityError
PhaseLexing
StatusReserved
Unit outcomeError: unit rejected. Warning: unit may still be published.
Canonical messagestray '|'; the disjunction operator is '||'

Cause

The lexer encountered one vertical bar not followed by another vertical bar.

ALAM defines logical disjunction as || only. It has no bitwise vertical-bar operator.

Example that triggers it

const Bool result = true | false;

Remedy

Replace the lone vertical bar with || when logical disjunction is intended.

Remove the operator when no disjunction was intended.
