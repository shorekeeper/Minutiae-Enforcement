Cause

This code is reserved for a parser diagnostic concerning an invalid record-construction target.

The current parser rejects or reinterprets invalid construction targets through other parser and elaboration paths and does not emit P005.

Example that triggers it

No triggering example is defined for this diagnostic.

Remedy

No action is required for the current compiler.

For record construction, place a constructible record name, constructor path, or supported Step or Rule instantiation immediately before the opening brace.
