List type

type

A homogeneous ordered collection type whose element type is invariant.

Syntax

list-type =
    element-type "[" "]" ;

list-literal =
    "{"
        [ expression
          { "," expression }
          [ "," ] ]
    "}" ;

Parameters

PartTypeDescription
Element type Any value type Exact type required of every list item.
Items Element type Values retained in source order.

Constraints

  • All items must have one identical element type. List element types are invariant.
  • A non-empty list may infer its element type from its first item.
  • An empty list requires an expected list type from its surrounding field, parameter, or declaration.
  • ALAM has no heterogeneous tuple type. Values of unrelated types cannot share one list.
  • The current language exposes no indexing, slicing, concatenation, insertion, or iteration expression.

Semantics

Items are evaluated from left to right and retained in source order. A list is immutable after construction.

Lists primarily serve total record fields and compile-time tables. Examples include rule citations, annotations, escalation rungs, and sequence steps.

A matrix is exposed to expressions as a list of its schema record type. expand is the dedicated element-wise operation over such a value.

Under the Java implementation

The static type is Type.ListT(Type element). Type equality and assignability require exact element-type identity. A list of Chat is not assignable to a list of base Event, even though a single Chat value is event-assignable to Event.

The value representation is Value.ListV(List<Value>). Evaluators create a mutable list while visiting items, then publish an immutable copy.

Example

const Rule[] cited_rules = {
    P.2.1,
    P.2.4,
};

const Annotation[] flags = {
    Annotation::Notify("staff"),
    Annotation::Evidence,
};

Diagnostics

P001A list delimiter or type-array closing bracket is missing.
E003A list item has a different element type or the list type is incompatible with its position.
E033An empty list appears where no expected element type is available.
