Conventions used in these docs

<angle> A required argument you replace with a value.
[square] An optional argument.
a|b Mutually exclusive alternatives.
P.3.2 A rule identifier from your rules.yml. Examples use the shipped default rule set.
1h, 7d, 1w2d Durations: seconds, minutes, hours, days, weeks; segments combine. permanent is a distinct literal, not a duration.
