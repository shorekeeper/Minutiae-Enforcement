MEMinutiae Enforcement docs

Права

Дерево узлов намеренно мелкозернистое, но взаимодействуете вы с ним через два бандла. Прочтите первую секцию, даже если остальное пролистаете: она объясняет поведение, которое регулярно удивляет операторов.

Как работают операторские значения по умолчанию

Каждый листовой узел регистрируется с default: false. Зарегистрированное false действует и на операторов — это стандартное поведение Bukkit, а не причуда плагина. Оператор получает лист только через цепочку гранта:

minutiae.admin (default: op)
  -> minutiae.staff: true
       -> minutiae.command.enforce: true

Следствия, которые стоит усвоить:

  • Если команда исчезла у оператора — цепочка разорвана, обычно дублирующимся ключом в правленом вручную paper-plugin.yml, который YAML разрешает, молча оставляя только последнее вхождение.
  • Плагины прав, отрицающие minutiae.admin для группы, обрывают всё под ним разом — это же и штатный способ построить собственный уровень доступа.
  • После смены операторского статуса или грантов бандла попросите игрока перезайти: дерево команд клиента не всегда пересылается при живой смене прав.

Два бандла

minutiae.staff Повседневная модерация: выдача, снятие (все три вида, включая чужие санкции), дела, разбор апелляций, диагностика и повседневные аннотации. default: false; выдайте его группе модераторов.
minutiae.admin Всё из staff плюс разрушительная поверхность и персонация: массовые снятия, откат снятий, снятие банов, каскады, изменения санкций, отягчающая коммутация, @as, @backdate, @shadow, перезагрузки. default: op.

С LuckPerms:

/lp group moderator permission set minutiae.staff true
/lp group admin permission set minutiae.admin true

Узлы снятия

Семейство снятия делится по последствиям, а не по командам; модель за каждым узлом разобрана на странице Снятие санкций.

minutiae.lift.pardon Снятие видом pardon. В staff.
minutiae.lift.served Снятие видом served. В staff.
minutiae.lift.foreign Снятие санкции, выданной кем-то другим. В staff; отзовите его у младшего уровня, которому положено отменять только свою работу.
minutiae.lift.blocking Снятие временного или перманентного бана. По умолчанию только админ.
minutiae.lift.cascade Снятие целой цепочки дел. Только админ.
minutiae.lift.bulk Массовое снятие по правилу или модератору. Только админ.
minutiae.lift.reinstate Откат ошибочного снятия. Только админ.

Узлы аннотаций

Каждая аннотация требует minutiae.annotation.<name> — включая аннотации из шаблона, так что шаблон не выдаст модератору флаг, которого нет у его группы. Бандл staff несёт повседневный набор; четыре узла зарезервированы за админом:

minutiae.annotation.shadow Скрытую изоляцию легко использовать во вред и трудно заметить.
minutiae.annotation.as Атрибуция другому имени — персонация по замыслу.
minutiae.annotation.backdate Переписывание хронологии требует доказательств и старшинства.
minutiae.annotation.remand Меры пресечения обходят обычную дисциплину мер.

Направленные и отдельные узлы

minutiae.commute.mitigate Коммутировать меру шаблона в сторону более лёгкой. В staff.
minutiae.commute.aggravate В сторону более тяжёлой. Только админ — усиление строгости требует старшинства.
minutiae.waive Снять требование компаньона из матрицы связей. Только админ.
minutiae.amend Изменить существующую санкцию. Только админ.
minutiae.appeal.submit Подать апелляцию. default: true — есть у каждого игрока.
minutiae.appeal.review Разбирать и решать апелляции. В staff.
minutiae.notify.staff Получать уведомления канала staff. В staff.
minutiae.fingerprint.alert Получать оповещения об обходе банов. В staff.
minutiae.rank.* Не узлы возможностей: маркеры, потребляемые секцией атрибуции рангов конфигурации для подписи объявлений. Выдавайте их по группам; они ничего не открывают.

Рецепты

Младший уровень хелпера

Поверхность staff, но отменять можно только свои небановые санкции, и без скрытого инструментария:

/lp group helper permission set minutiae.staff true
/lp group helper permission set minutiae.lift.foreign false
/lp group helper permission set minutiae.annotation.ghost false

Старший модератор со снятием банов

/lp group srmod permission set minutiae.staff true
/lp group srmod permission set minutiae.lift.blocking true

Аудитор только для чтения

/lp group auditor permission set minutiae.command.info true
/lp group auditor permission set minutiae.command.docket true
/lp group auditor permission set minutiae.command.rules true
Примечание

Автоматизация (движок ALAM) и веб-панель не используют игровые права: каждая несёт собственный ограниченный набор грантов в config.yml, так что ни одна не превысит явно настроенного вами — даже при компрометации токена или определения.
