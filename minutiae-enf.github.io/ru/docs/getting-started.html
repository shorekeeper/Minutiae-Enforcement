MEMinutiae Enforcement docs

Начало работы

От jar-файла до первой аудируемой санкции. Ничего здесь не требует сервера баз данных или внешних зависимостей.

Требования

Сервер Paper 1.21 или новее. Spigot не поддерживается: плагин напрямую использует API команд и чата Paper.
Java Java 21 или новее.
Хранилище Устанавливать нечего. Плагин поставляется с SQLite и создаёт файл базы при первом запуске.

Установка

  1. Положите MinutiaeEnforcement.jar в каталог plugins/ вашего сервера.
  2. Запустите сервер. Лог запуска печатает по строке на подсистему; здоровый запуск заканчивается строкой ready: all subsystems online.
  3. Остановите сервер, если хотите отредактировать сгенерированную конфигурацию до боевого использования, или продолжайте с умолчаниями.

Первый запуск создаёт следующие файлы:

config.yml Рабочая конфигурация. Каждая секция документирована прямо в файле; обзор — на странице Конфигурация.
rules.yml Правила вашего сервера нумерованными записями. Поставляется с примером из девяти правил; замените их своими.
layouts.yml Шаблоны санкций. Поставляется с примерами для гриферства, читов и токсичного чата.
lang/en.yml Языковой пакет для вашей локали по умолчанию. Отредактированные файлы никогда не перезаписываются; ещё не слитые ключи берутся из встроенных шаблонов.
minutiae.db База SQLite. Здесь живут санкции, апелляции, аудит, стенограммы чата и данные отпечатков.
alam/ Определения автоматизации. Пуст по умолчанию; не трогайте, пока не захотите автоматическое принуждение.

Проверка установки

/me-ban ping

Команда прощупывает хранилище и сообщает задержку и версию схемы. Если она отвечает — всё под ней работает.

Выдача прав

Два бандла покрывают почти всё:

minutiae.staff Повседневная модерация: выдача, снятие, дела, разбор апелляций, большинство аннотаций.
minutiae.admin Всё, включая массовые снятия, откат снятий, перезагрузки и разрушительные аннотации. Выдан операторам по умолчанию.
Примечание

Листовые узлы регистрируются с default: false, что действует и на операторов. Оператор получает лист только через цепочку minutiae.admin. Если команда исчезла у оператора — цепочка в paper-plugin.yml или в вашем плагине прав разорвана; см. Права.

С LuckPerms:

/lp group moderator permission set minutiae.staff true

Первая санкция

С поставляемым шаблоном:

/enforce SomePlayer ::toxic

Шаблон toxic ссылается на правило P.2.4, применяет трёхчасовой мут и несёт @warn-first(2): первые два нарушения по этому правилу становятся формальными предупреждениями вместо мута. Выполните дважды на тестовом аккаунте и посмотрите, как меняется отчёт.

Полностью вручную, без шаблона:

/enforce SomePlayer @measure(KICK) reason="Тест, игнорировать"

Пока не уверены? Просмотр без применения:

/enforce SomePlayer ::griefing @dry-run

Как отменить

Каждый отчёт заканчивается строкой ID. Снимайте по этому идентификатору:

/enforce lift 1 vacate "Тестовая санкция"

vacate стирает санкцию из прецедента игрока — то, что нужно для теста. Различие между аннулированием, помилованием и зачётом срока разобрано на странице Снятие санкций.
